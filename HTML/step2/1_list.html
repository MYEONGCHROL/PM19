순서가 없는 목록태그

  • 번호없는 목록1, 목록 표시 빈원
  • 번호없는 목록2, 목록 표시 찬원
  • 번호없는 목록3, 목록 표시 사각형

ul은 unordered list 의 약자다.
사전적 의미는 중요하지 않다.
속성값은 3개, circle, disc, square 가 있으나, 이것 역시 속성값을 외울 필요는 없다.

순서가 있는 목록태그

  1. 번호 있는 목록 1, 목록표시 대문자 알파벳
  2. 번호 있는 목록 2, 목록표시 소문자 알파벳
  3. 번호 있는 목록 3, 목록표시 대문자 로마숫자
  4. 번호 있는 목록 4, 목록푝시 소문자 로마숫자
  5. 번호 있는 목록 5, 목록표시 아라비아 숫자

ol은 ordered list 의 약자이다.
사전적 의미는 중요하지 않다.
속성값은 4개(+@), "A, a, I, i"가 있다.
중요한 것은 레이아웃을 구성 할때, 번호록 정렬 되어야 하는 목록 형태의 컨텐츠를 표현하는 레이아웃

HTML
Hyper Text Markup Language
XML
Extensible Markup Language
dl
definition list

dl 은 definition list의 약자이다.
정의형 목록인데, 마찬가지로 사전적 의미는 중요하지 않다.
특별한 속성값은 없다.
